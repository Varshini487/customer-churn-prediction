# 📉 Customer Churn Prediction

A **Machine Learning system** that predicts which customers are likely to churn using multiple classification algorithms.

## 📊 Model Performance
| Model | Accuracy | AUC-ROC |
|-------|----------|---------|
| Logistic Regression | 79% | 0.84 |
| Random Forest | 83% | 0.88 |
| XGBoost | **86%** | **0.91** |
| Neural Network | 84% | 0.89 |

## 🛠️ Tech Stack
- **Python, Pandas, NumPy** – data processing
- **Scikit-learn, XGBoost** – ML models
- **SHAP** – model explainability
- **Matplotlib / Seaborn** – visualizations
- **Streamlit** – deployment

## 🚀 Getting Started
```bash
git clone https://github.com/Varshini487/customer-churn-prediction
cd customer-churn-prediction
pip install -r requirements.txt
streamlit run app.py
```
